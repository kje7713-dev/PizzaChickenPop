# Xcode Project

This is a placeholder for the actual Xcode project file.

To create the actual Xcode project:

1. Open Xcode
2. File → New → Project
3. Choose "iOS" → "Game" template
4. Set Product Name to "PizzaChicken"
5. Choose SpriteKit as the Game Technology
6. Set the location to this directory
7. Add the existing Swift files from Sources/ to the project
8. Add the Resources/ folders to the project

The Xcode project file will be named `PizzaChicken.xcodeproj` and will appear in this directory.

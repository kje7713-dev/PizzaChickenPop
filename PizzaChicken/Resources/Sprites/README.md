# Sprites

This directory contains sprite images for the game.

## Expected Files

- `chicken_normal.png` - The chicken in its normal state
- `chicken_stuffed.png` - The chicken after eating some pizza
- `chicken_about_to_pop.png` - The chicken just before explosion
- `pizza.png` - The pizza sprite

## Notes

- Use PNG format with transparency
- Recommend @2x and @3x versions for retina displays
- Keep it cartoony and family-friendly

# Particle Effects

This directory contains SpriteKit particle effect files (.sks).

## Expected Files

- `FeatherExplosion.sks` - Feather particles for chicken explosion
- `CrumbBurst.sks` - Crumb particles for explosion effect

## Notes

- Create these files in Xcode using the SpriteKit Particle Emitter
- Use cartoony, family-friendly effects
- Keep particle counts reasonable for performance
